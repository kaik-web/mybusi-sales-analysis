# Dashboard Comercial — Mybusi

Projeto de análise comercial desenvolvido com SQL Server e Power BI.

## Visão geral
Dashboard interativo com análise completa de vendas de uma empresa fictícia de tecnologia.

## Métricas analisadas
- Receita total e ticket médio
- Quantidade de vendas e clientes
- Faturamento por ano e mês
- Vendas por gênero
- Faturamento por continente
- Faturamento por marca

## Ferramentas utilizadas
- SQL Server — modelagem e armazenamento dos dados
- Power BI — relacionamentos, medidas DAX e dashboard

## Principais insights
- Apple lidera o faturamento por marca com R$ 40 mil
- Europa é o continente com maior concentração de vendas
- Distribuição de vendas equilibrada entre gêneros (50/50)
- Pico de faturamento em julho de 2022 com R$ 19.497
